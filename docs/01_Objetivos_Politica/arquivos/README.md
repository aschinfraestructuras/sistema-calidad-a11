## 🚀 Como Adicionar Arquivos

1. **Colocar arquivo** na pasta `docs/01_Objetivos_Politica/arquivos/`
2. **Fazer deploy** com `npx vercel --prod`
3. **Arquivo fica online** automaticamente
4. **Acesso direto** via URL
