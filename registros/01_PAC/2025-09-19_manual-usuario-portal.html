Manual de Usuario

Portal de Calidad ASCH-OHLA - Sistema de Gestión de Calidad

UTE A11 LANGA-ARANDA

1. Introducción

Este manual describe el funcionamiento del Portal de Calidad ASCH-OHLA, un sistema integral de gestión de documentación técnica para la obra A-11.

Objetivo: Centralizar y organizar toda la documentación de calidad en un sistema accesible y funcional.

2. Acceso al Sistema

El portal está disponible en:

https://sistema-calidad-a11.vercel.app

Credenciales de Acceso:

  • Administrador: admin / asch2025
  • Usuario Calidad: calidad / calidad2025
  • Usuario Obra: obra / obra2025

3. Funcionalidades Principales

3.1 Dashboard

  • Vista general de estadísticas del proyecto
  • Indicadores de progreso por capítulo
  • Documentos subidos recientemente
  • Acciones rápidas

3.2 Gestión de Documentos

  • Subida de documentos (HTML, PDF, Word, Excel, etc.)
  • Visualización integrada de documentos
  • Descarga e impresión
  • Eliminación de documentos subidos

3.3 Organización por Capítulos

El sistema organiza la documentación en 21 capítulos:

  • 01 - Sistema de Gestión de Documentos
  • 02 - Plan de Ensayos y Controles
  • 03 - Objetivos y Política de Calidad
  • 04 - Programación y Comunicaciones de Obra
  • 05 - Trazabilidad de Materiales
  • 06 - Puntos de Inspección y Control (PPI)
  • Y 15 capítulos adicionales...

4. Búsqueda y Filtrado

Utilice la barra de búsqueda en la parte superior para encontrar documentos por:

  • Título del documento
  • Descripción
  • Tags o etiquetas
  • Código de referencia

5. Tipos de Archivos Soportados

  • HTML - Documentos web editables
  • PDF - Documentos de texto
  • Word (.doc, .docx)
  • Excel (.xls, .xlsx)
  • Imágenes (JPG, PNG, GIF, etc.)
  • Archivos de texto (.txt, .rtf)
  • Y muchos más formatos
